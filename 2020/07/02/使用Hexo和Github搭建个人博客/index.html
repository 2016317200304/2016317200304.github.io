使用Hexo和Github搭建个人博客

在Windows系统上搭建个人博客，这是第一篇博客，也是搭建步骤

环境配置

安装 git、node.js、hexo

  1. 安装git，网上教程很多，随便找，我是以前安装过，就不重复一次了

  2. 安装node.js，官网直接下载，也可以查看网上的教程，很多，安装过就不重复了
    查看node的版本打开cmd命令行输入
    node -v
    出现版本号即已安装node

  3. 安装hexo
    cmd命令行里输入
    npm install -g hexo-cli
    当然先按照淘宝的cnpm再用cnpm安装hexo也可以，但是这里没有必要，npm安装hexo挺快的，几分钟的事

    查看hexo版本
    hexo -v

环境配置完成了

搭建本地博客

创建一个文件夹Blog(文件夹名随意)，在文件夹里右键单击打开git bash here，接下来在git bash here 的界面操作就可以

hexo初始化，输入hexo init，耐心等待，这个过程需要几分钟

然后就可以在本地Blog文件夹中看见生成了多个文件和文件夹

继续在git的界面，输入
hexo s
出现以下内容代表成功了

浏览器打开http://localhost:4000
可以看到最初的自动生成的博客模板，内容事hexo使用教程

写新的博客可以根据教程中的hexo new "my new blog"生成并编辑，不过我觉得挺麻烦，就尝试了一下直接在VSCode上写一个MarkDown文件，写好了直接复制到/Blog/source/_posts文件夹里

然后清理一下
hexo clean

生成静态页面
hexo g

启动本地服务器
hexo s

再浏览器打开http://localhost:4000，可以看到新的博客了
